# Flask Contact App
A simple Flask web application with a homepage and contact form
## Tech Stack
- Backend: Python  (FLask)
- Frontend: HTML
## Features
- Displays a welcome message
- Simple contact form to collect user input
## How to Run
1. Install Flask :
pip install flask
2. Run the app :
 python app.py 
3. Open browser:
http://127.0.0.1:5000
